강원도 반려동물
동반 관광정보 API 시스템

강원도내 반려동물 동관 관광 여행 데이터를 제공합니다.
누구나 쉽게 검색이 가능하고 확인된 데이터는 자유롭게 사용이 가능합니다.

API 사용방법

강원 반려동물 동반 관광정보 API시스템은 관광기관 최초로 반려동물 동반 가능 관광DB를 구축한 시스템입니다.
기존 반려동물 동반 가능여부의 분산된 정보를 검증된 자료조사를 통해 제공하고 있습니다.
제공된 JSON을 통해 데이터를 무료로 사용 하실 수 있습니다.
