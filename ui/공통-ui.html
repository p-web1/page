  • 굵은 서체(bold) skinFont-bold
  • 보통 굵서체(normal) skinFont-normal
  • 붉은색 서체(#d70808) skinColor-red
  • 푸른색 서체(#09bc9) skinColor-blue
  • 밑줄 추가 skinTxd-UL

  • 텍스트 오른쪽 정렬 skinTxa-right

  • 텍스트 가운데 정렬 skinTxa-center

  • 텍스트 왼쪽 정렬 skinTxa-left

여백속성

- 클래스로 여백 속성을 준다. (자세한 목록은 common.css의 css element로 검색하면 있습니다.)

여백 속성의 네이밍 규칙

  • skin + 속성위치 + 값
  • 값은 0, xsmall(10px), small(20px), medium(30px), large(40px), xlarge(50px)로 이루어져 있다.
  • 예) skinMt-0 : 상단 마진값 0
  • 예) skinPdr-medium : 우측 패딩값 30px
  • 예) skinPdl-0 : 왼쪽 패딩값 0
